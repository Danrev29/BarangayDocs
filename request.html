How it Works

Get started with these 3 easy steps:

1

Document Request


The user logs in, selects the document, fills out the form, and submits the request.

2

Processing & Approval


Barangay staff reviews, verifies, and processes the request. Once approved, the system will notify the user.

3

Pickup & Pay


The user visits the Barangay office to pick up the document.
